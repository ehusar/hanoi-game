# hanoi-game
A classic Tower of Hanoi game

Puzzle Games
A small collection of browser-based puzzle games. Each game is a single self-contained HTML file — no install, no build step. Just download and open in a browser, or view live via GitHub Pages.
Games
Tower of Hanoi — Inked
A cel-shaded, 3D-tilted take on the classic disk-stacking puzzle.
How to play
	•	Click a peg to lift its top disk, then click another peg to drop it there.
	•	Move the entire stack from peg A to peg C, one disk at a time.
	•	A bigger disk can never rest on a smaller one.
	•	Get it wrong and the tower resets — no partial credit, you start fresh.
	•	Watch for a message at 15 moves and again at 30 moves.
	•	Your best solve is tracked for the current session.
